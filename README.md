# Odoo 18 Custom Addons

Custom Odoo 18 modules showcasing business workflows, approvals, and automation. A collection of custom Odoo modules built for real-world business use cases. Professional Odoo 18 custom development portfolio (Python, ORM, Workflows). Custom-built Odoo modules including approvals, reports, and automation logic.
