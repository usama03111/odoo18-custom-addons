# CRM Social & Industry Fields

**Category:** CRM
**Version:** 18.0.1.0.0
**Author:** Usama Wazir

## Summary
Adds Industry Type and Social Media fields (Facebook, Instagram, LinkedIn, TikTok) to CRM Leads/Opportunities

## Description
CRM Social & Industry Fields
============================
This module adds additional fields to the CRM Lead/Opportunity form:
- Industry Type  
- Facebook URL  
- Instagram URL  
- LinkedIn URL  
- TikTok URL  

All social fields use URL widgets with helpful placeholders and appear right after the Website field.
