Odoo Whatsapp Connector
=======================
Odoo Whatsapp connector for Sales, Invoice, and a floating button on the Website, along with icons for Whatsapp and email in the systray bar. **Now with webhook support to receive and display customer replies!**

Features
========
* Send WhatsApp messages from Sales Orders and Invoices
* Floating WhatsApp button on website
* WhatsApp and Email icons in systray bar
* **NEW: Receive customer replies via webhook**
* **NEW: Chat-like interface to view conversations**
* **NEW: Message history storage**
* **NEW: Automatic customer linking**

Configuration
=============
1. Go to **Settings > General Settings > WhatsApp Integration**
2. Enable "Enable WhatsApp API Integration"
3. Select your API provider (Twilio or WhatsApp Business API)
4. Configure the webhook URL in your API provider
5. Test the integration using the provided test script

For detailed setup instructions, see: `doc/WHATSAPP_WEBHOOK_SETUP.md`_
